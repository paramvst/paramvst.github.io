Setup Your Personal MongoDB Server

MongoDB is a popular document-oriented database system that is widely used in web development. If you are new to MongoDB and want to start using it, this guide will help you set up your own MongoDB server.

Prerequisites

  • A computer running Windows, macOS, or Linux
  • Administrative access to install software on your computer
  • An Internet Connection (just while Downloading)
  • A web browser

Step 1. Download and install MongoDB

The first step is to download and install MongoDB. You can download the latest version of MongoDB from the official website https://www.mongodb.com/.

Once you have downloaded the installer follow the steps to start the installer.

  • Double-click the installation package to start the installer.
  • Follow the on-screen instructions to install MongoDB. Make sure you select the option to install the MongoDB server and the MongoDB Compass tool.
  • Once the installation is complete, you should see a message indicating that MongoDB has been successfully installed.

Step 2. Configure MongoDB

Once you have installed MongoDB, you need to start the MongoDB server. Here's how to do it:

  1. Open a command prompt or terminal window.
  2. Navigate to the MongoDB installation directory.
  3. Create a directory to store the MongoDB data files: mkdir data.
  4. Start the MongoDB server: mongod --dbpath=data.

After running the mongod command, you should see output similar to the following:

2022-03-21T13:45:23.312+0000 I CONTROL [initandlisten] MongoDB starting : pid=12345 port=27017 dbpath=data 64-bit host=your-host-name

This means that the MongoDB server is running and listening on port 27017.

Conclusion

Your Mongo DB Server is now set and now you would require a program or platform to connect with this server, It could be via mongo Shell, Mongo Compass software, or any other software like Java, Node, Python to access your data.
