Connect MongoSh with Mongo Server

Prerequisites:

Before you begin, you will need to have the following:

  • A MongoDB server set up and running.
  • The MongoDB command-line interface, or Mongo shell, installed on your computer. You can download it from the official MongoDB website.

Step 1: Open the Command Line

To connect to your MongoDB server using the Mongo shell, you will need to open a command line interface on your computer. This can be done by opening the Command Prompt on Windows, Terminal on macOS, or your preferred command-line interface on Linux.

Step 2: Connect to the MongoDB Server

Once you have your command line open, you can connect to your MongoDB server by running the following command:

mongo

This will start the Mongo shell and connect to the default MongoDB server running on your local machine.

If you have set up your MongoDB server on a remote machine or with a different port or username/password, you can connect to it using the following command:

mongo --host <host_name>:<port_number> --username <username> --password <password> --authenticationDatabase <database_name>

Replace <host_name> with the name or IP address of the machine where the MongoDB server is running, <port_number> with the port number that the MongoDB server is listening on, <username> and <password> with your MongoDB username and password, and <database_name> with the name of the authentication database.

Step 3: Switch to a Database

Once you are connected to your MongoDB server, you can switch to a specific database using the following command:

use <database_name>

Replace <database_name> with the name of the database you want to connect to.

Step 4: Run MongoDB Commands

Now that you are connected to a database, you can run MongoDB commands using the Mongo shell. For example, to show all the collections in the current database, you can run the following command:

show collections

To create a new collection, you can run the following command:

db.createCollection("<collection_name>")

Replace <collection_name> with the name of the collection you want to create.

Step 5: Exit the Mongo Shell

To exit the Mongo shell, you can run the following command:

exit

This will disconnect from the MongoDB server and close the Mongo shell.

That's it! You now know how to connect the Mongo shell with your MongoDB server. With this knowledge, you can start exploring and working with your MongoDB databases using the command line.
