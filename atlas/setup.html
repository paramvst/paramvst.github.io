Atlas Setup

Why use MongoDB Atlas?

MongoDB Atlas is a cloud-hosted service that provides fully managed MongoDB databases. It allows you to scale your database easily, manage it from anywhere, and provides built-in security features to protect your data.

Getting started with MongoDB Atlas

To get started with MongoDB Atlas, you'll need to sign up for an account. Head to the MongoDB Atlas website and click the "Sign Up" button in the top-right corner.

Once you've signed up, you'll need to create a new project. A project is a way to organize your MongoDB databases, users, and other resources. Click the "Create a New Project" button and give your project a name.

Creating a MongoDB database

After creating a project, you'll need to create a MongoDB database. Click the "Build a New Cluster" button, choose a cloud provider and region, and select a cluster tier. You can choose a free tier to get started.

Once the cluster is created, you'll see a "Connect" button. Click it and select "Connect Your Application". This will provide you with a connection string that you can use to connect to your MongoDB database from your application.

Add users who can access your MongoDB database

You can also add users to your database by clicking the "Database Access" tab and then clicking the "Add New User" button. Give your user a name and password, and specify the database(s) they should have access to.

How to create a database in MongoDB database

  1. Log in to your MongoDB Atlas account and go to the "Clusters" section.
  2. Select the cluster you want to create the database in.
  3. Click on the "Collections" button in the left-hand menu.
  4. Click on the "Create Database" button.
  5. Enter a name for your new database.
  6. Click the "Create" button to create the new database.

Once you've created a collection, you can add data to it by clicking the "Insert Document" button. This will open up a JSON editor where you can enter your data.

Doing CRUD operations on the data

CRUD stands for Create, Read, Update, and Delete, which are the four basic functions of persistent storage. MongoDB provides several methods to perform CRUD operations on the data stored in the database.

Create

To create a new document in a collection using MongoDB Atlas, you can follow these steps:

  1. Log in to your MongoDB Atlas account and select the project where your cluster is located.
  2. Click on the "Collections" tab in the left-hand navigation menu and select the collection where you want to create a new document.
  3. Click on the "Insert Document" button in the upper-right corner of the screen.
  4. Enter the fields and values for the new document in the form that appears, and then click on the "Insert Document" button to save it.

Read

To read documents from a collection using MongoDB Atlas, you can follow these steps:

  1. Log in to your MongoDB Atlas account and select the project where your cluster is located.
  2. Click on the "Collections" tab in the left-hand navigation menu and select the collection where you want to read documents from.
  3. Click on the "Find" button in the upper-right corner of the screen.
  4. Enter a query in the form that appears to specify which documents you want to find, and then click on the "Find" button to execute the query.

Update

To update a document in a collection using MongoDB Atlas, you can follow these steps:

  1. Log in to your MongoDB Atlas account and select the project where your cluster is located.
  2. Click on the "Collections" tab in the left-hand navigation menu and select the collection where you want to update a document.
  3. Find the document you want to update, and then click on the "Edit" button next to it.
  4. Make the necessary changes to the document in the form that appears, and then click on the "Save Changes" button to them.

Delete

To delete a document from a collection using MongoDB Atlas, you can follow these steps:

  1. Log in to your MongoDB Atlas account and select the project where your cluster is located.
  2. Click on the "Collections" tab in the left-hand navigation menu and select the collection where you want to delete a document from.
  3. Find the document you want to delete, and then click on the"Delete" button next to it.
  4. Confirm that you want to delete the document by clicking on the "Delete Document" button in the confirmation dialog that appears.

Wrapping up

That's it! You now have a basic understanding of how to set up MongoDB Atlas and start working with your MongoDB database. From here, you can explore more advanced features like sharding, backups, and monitoring.
